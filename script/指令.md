# 如何使用指令
在遊戲中導入好腳本後在聊天框輸入以下指令來運行
&help
-- 指令幫助

&fly
-- 關閉/開啟飛行

&god
-- 關閉/開啟無敵

&kick {player_id}
-- 踢掉玩家

&add {player_id} {id} {num}
-- 添加物品

&kill {player_id}
-- 殺死玩家

&checkbox {player_id} {range}
-- 設置玩家拾取範圍

&tp {player_id} {x} {y} {z}
-- 設置玩家位置

範例

&add 20334027 10500 1
給迷你號為20334027的玩家添加一個地形編輯器

&fly
開啟飛行(僅自己
